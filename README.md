# DentakuWebApp

C# 製のシンプルなコンソール電卓です。四則演算（足し算・引き算・かけ算・割り算）に対応し、メニュー形式で操作できます。

## 必要環境
- .NET 8.0 SDK 以降

## 使い方
1. リポジトリのルートで依存関係を復元し、ビルドします。
   ```bash
   dotnet restore
   dotnet build
   ```
2. アプリを実行します。
   ```bash
   dotnet run
   ```
3. 表示されるメニューから操作を選び、数値を入力してください。

## 主な機能
- メニューで操作を選択し、数値を2つ入力するだけで計算結果を表示
- 無効な入力（数値以外、メニュー外の番号）に対する再入力要求
- 0 で割ろうとした際のエラーメッセージ表示
